 Average Time Spent by Users on Social Media
 Project Overview

This project analyzes a dataset that tracks how much time users spend on social media and the factors that influence their usage.

The goal is to:

Explore user behavior
Build predictive models
Extract insights about social media engagement

This project was developed as part of my Artificial Intelligence studies.

 Objectives
Perform Exploratory Data Analysis (EDA)
Understand the relationship between demographic and financial factors and social media usage
Build a Regression Model to predict time spent
Visualize and interpret insights for decision-making
 Dataset Features

The dataset includes user information such as:

Age
Gender
Country / Location
Profession
Income
Interests
Social Media Platform
Home Ownership
Car Ownership
Debt Status
 Exploratory Data Analysis (EDA)

Key findings from EDA:

Most users belong to middle-aged groups
Income variations influence time spent online
Users are distributed across urban, suburban, and rural areas

 Key Insights
Age, income, and platform choice strongly affect engagement.
Middle-aged and urban users are the most active.
Higher-income users tend to spend less time on social media.
Lifestyle and interests influence usage behavior.
Results can help businesses improve targeted marketing strategies.
